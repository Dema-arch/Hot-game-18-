# Інструкції для Git (якщо хочеш використовувати командний рядок)

# 1. Встанови Git з https://git-scm.com/download/win (якщо ще не встановлений)

# 2. Відкрий PowerShell в папці hot-questions-game і виконай:

git init
git add .
git commit -m "Add hot games"
git branch -M main
git remote add origin https://github.com/твій-username/hot-games.git
git push -u origin main

# Замість 'твій-username' та 'hot-games' підстав свої дані
