# Жаркі Ігри 🔥

Збірка інтерактивних ігор для пар та компаній.

## Ігри в збірці:

1. **Жаркі Питання** - 100 пікантних питань для пари + 100 для тусовки
2. **Правда чи Дія 18+** - 100 відвертих питань + 100 сміливих завдань
3. **Каверзні Дії** - 100 інтимних завдань для двох
4. **Фанти для Пари** - 100 романтичних та пікантних фантів
5. **Spin the Bottle** - класична гра для компанії з 4 людей
6. **Я Ніколи Не...** - 100 каверзних тверджень

## Як запустити локально:

Просто відкрийте файл `index.html` у браузері.

## Як викласти на хостинг:

### Варіант 1: Netlify (найпростіше)

1. Зареєструйтесь на https://www.netlify.com (безкоштовно)
2. Перетягніть папку `hot-questions-game` на сайт Netlify
3. Отримаєте посилання типу `https://your-site.netlify.app`

### Варіант 2: GitHub Pages

1. Створіть репозиторій на GitHub
2. Завантажте всі файли
3. У налаштуваннях репозиторію увімкніть GitHub Pages
4. Отримаєте посилання типу `https://username.github.io/repo-name`

## Технології:

- HTML5
- CSS3
- Vanilla JavaScript
- Адаптивний дизайн для мобільних пристроїв
